# プロトタイプ用ノートブック
このディレクトリには，ソルフェージュ生成の実験用コードを記述したノートブックを格納しています．

## ノートブック一覧
- `audio_mixing.ipynb`: 異なる音源の合成実験
- `crepe_transcription.ipynb`: torchCREPEによるボーカルのMIDI変換(うまくいかず)
- `midi_transcription.ipynb`: **basic-pitchによる特徴抽出と各種パイプラインによるボーカルの採譜**
- `music_separation.ipynb`: Demucsによるパート分離の実験
- `separation_by_mdx`: MDXNetによるボーカル分離や，ハモリ除去・リバーブ除去に関する実験
- `tuning_correction.ipynb`: チューニング補正に関する実験(音質低下のため，リサンプリング方式に変更済み)
